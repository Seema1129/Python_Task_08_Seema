Task Objective

The objective of this task was to learn and implement Object-Oriented Programming (OOP) concepts in Python by developing a menu-driven Employee Management System. The project demonstrates the use of classes, objects, constructors, methods, lists, file handling, and basic data management operations.

Project Description

The Employee Management System is a console-based application developed using Python. It allows users to manage employee records efficiently through a simple menu-driven interface. Employee information such as ID, Name, Department, Designation, and Salary is stored using objects of the Employee class.

Features Implemented
Created an Employee class using a constructor (__init__).
Added new employee records.
Displayed all employee records in a tabular format.
Searched employees by Employee ID or Employee Name.
Updated employee details such as Department, Designation, and Salary.
Deleted employee records using Employee ID.
Displayed salary statistics including:
Highest Salary
Lowest Salary
Average Salary
Total Number of Employees
Exported employee data to employees.csv.
Read employee data from the CSV file and displayed it.
